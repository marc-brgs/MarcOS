Le dahlia, célébré par Verlaine dans son 30ème poème saturnien, " … Ainsi le Dahlia, roi vêtu de splendeur, Elève sans orgueil sa tête sans odeur…", compte plus de 40 000 variétés qui se déclinent dans toutes les teintes possibles, excepté… le bleu. Ses fleurs, même pour les plus petites variétés (les"pompons"), sont toujours géréreuses et charnues. Conseils de Florazar pour la mise à l'eau : le dahlia est une fleur très éphémère. Les plus grands, à la tige plus robuste, tiennent pratiquement une semaine. Les feuilles des dahlias font pourrir les fleurs, il faut donc bien toutes les enlever et remplir le vase de très peu d'eau. Quand les dahlias sont intégrés dans un bouquet, les ôter dès qu'ils fanent pour qu'ils n'intoxiquent pas les autres fleurs.
