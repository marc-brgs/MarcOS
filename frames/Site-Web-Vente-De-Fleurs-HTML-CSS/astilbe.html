L'Astilbe est originaire d'extrême Orient. Les fleuristes de Florazar aiment l'insérer dans leurs bouquets pour leur donner une note vaporeuse et légère. Les inflorescences en épis peuvent prendre des nuances rouge-carmin, blanches ou violettes. Son feuillage bien découpé est également très décoratif. Conseils de Florazar pour la mise à l'eau : changer l'eau régulièrement et comme toutes les fleurs des champs, bien nettoyer la tige de ses feuilles pour que l'eau ne se salisse pas trop.
